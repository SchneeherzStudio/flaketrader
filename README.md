# FlakeTrader
FlakeTrader Discord Bot for live trading on realtime market with fictional currency

# About FlakeTrader
FlakeTrader uses 
- discord.js
- yahoo-finance2
- chartjs-node-canvas
- PostgreSQL
- pm2

# Commands
- /help (shows all commands of the bot)
- /info (shows information of the bot and used external npm packages)
- /intro (gives an short introduction on use of bot and trading)
- /status (temporary command, shows if /commands are available or issued)
- /daily (claims an daily amount of coins for use in trading)
- /buy (used to buy an asset of the market)
- /sell (used to sell the bought assets)
- /search (search the symbol of the asset for lookup in /market)
- /market (lookup the course of an asset in 1D, 3D, 5D, 1W, ...)
- /balance (shows the total money (cash, investments) and +/- % on assets)
- /top (provides leaderboard of top money on server)
- /global (provides global leaderboard of money)
